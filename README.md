# Trabalho-Faculdade.
Sistema desenvolvido em Python para geração de orçamentos de aluguel de imóveis.

## Funcionalidades
- Cálculo do aluguel para apartamentos, casas e estúdios.
- Acréscimo de quartos, garagem e vagas de estacionamento.
- Desconto para apartamentos sem crianças.
- Parcelamento do contrato imobiliário.

## Como executar

1. Instale o VSCODE
2. Baixe ou clone este repositório.
